Q5. Define Favicon and give an illustrative example.
Answer:- A favicon (favourite icon)browser's address bar is a tiny icon included along with a website, which is displayed in places like the page tabs and bookmarks menu.Usually, a favicon is 16x 16 pixels in size and stored in the GIF, PNG, or ICO file format.They are used to improve browser's address bar, for user experience and enforce brand consistency. When a familiar icon is seen in the example, it helps users know they are in the right place.
